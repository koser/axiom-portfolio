# Tensile Networks
**Date:** 2026-03-16
**Project ID:** AG21
**Library:** p5.js

## Concept
An animated field of interconnected organic filaments that bulge, constrict, and pulse like cellular membranes under tension. The piece translates the moodboard's flowing vertical line networks into a living system where curves are never static — each strand is a chain of vertices displaced by layered noise fields, creating wobbly, irregular forms that breathe and undulate. The visual logic mirrors biological systems under microscopic observation: tension, elasticity, and the constant renegotiation of space between neighboring forms.

## Palette
#fafaf8 (background — off-white canvas), #0d0d0d (primary — black sinuous strands), #1a1a1a (secondary — slightly lighter shadow strands for layering depth), #f2f2f0 (accent — rare hairline highlights on bulges)

## Composition
The canvas is divided into 7-9 vertical columns spanning full height, each column containing 2-4 overlapping strand chains. Strands begin as vertical paths but deviate via noise displacement, creating the organic bulging seen in the moodboard. Composition shifts over time: Phase 1 establishes evenly-spaced vertical rhythm with subtle movement. Phase 2 introduces lateral drift — columns slowly migrate left/right by 50-120px over 15 seconds, creating density clusters in quadrants then dispersing. Phase 3 creates voids — strands retract from center, concentrating at left and right thirds, leaving negative space that emphasizes the remaining forms. Edge behavior: strands that exit viewport wrap to opposite side with randomized vertical offset.

## Elements
STRAND CHAINS: constructed using curveVertex() with 40-60 control points per strand. Each control point's x-position is displaced by multi-octave Perlin noise (2 octaves: base frequency 0.003, detail frequency 0.015) creating irregular bulges 20-80px wide. Y-positions are evenly spaced at 12-18px intervals. Stroke weight varies along length using sin() mapped to distance from top: 1.5-3.5px, creating taper effect. Each strand has 'tension parameter' (0.4-0.9) controlling maximum lateral displacement. CLUSTER NODES: at intersection points where strands approach within 15px, draw small irregular blobs using beginShape() with 8-12 vertices arranged in circle, each vertex displaced radially by noise() 3-8px. These nodes have 60% opacity and appear/disappear based on proximity threshold. BACKGROUND TEXTURE: 600-800 ultra-thin (0.3px) bezier curves scattered across canvas, nearly invisible (#f0f0ee), providing subtle organic texture layer behind main strands. HIGHLIGHT TRACES: occasional (5-8 total) hairline curves (0.8px, #f2f2f0) that follow strand paths with 2-4px offset, appearing only where strand curvature exceeds threshold (derivative angle > 15 degrees), emphasizing dramatic bulges.

## Animation
Phase 1 (0-12s): All strand control points animate using time-evolving noise: noiseDetail(2, 0.5), noise input is (x * 0.003, y * 0.003, frameCount * 0.0008). This creates slow undulation at 0.4-0.8px/frame lateral movement per control point. Cluster nodes fade in/out with 90-frame (1.5s) sinusoidal easing as strands drift into/out of proximity. Stroke weight oscillates globally using sin(frameCount * 0.002) * 0.4, adding breathing quality. Phase 2 (12-35s): Introduce lateral column drift. Each column's base x-offset shifts using Perlin noise at frequency 0.0005 * frameCount, creating slow 0.3-0.6px/frame horizontal migration. Maximum drift is clamped at ±120px from origin. Strand undulation speed increases: noise time parameter becomes frameCount * 0.0015 (nearly 2x faster). Some strands (30%) begin counter-phase movement where noise offset inverts, creating visual tension between neighboring strands moving opposite directions. Cluster nodes now pulse scale from 0.8-1.3 over 120 frames using ease-in-out cubic function. Phase 3 (35s+): Initiate spatial redistribution. Center 40% of canvas becomes 'repulsion zone' — strands whose base x-position falls in this zone receive outward force pushing them toward nearest third (left or right). Force is 1.2px/frame with exponential easing over 240 frames. Remaining strands in outer thirds increase density, creating concentrated vertical forests. Undulation becomes extreme: noise detail increases to (3, 0.6), some control points can displace up to 100px laterally. Every 8-12 seconds, random strand undergoes 'tension release' — all control points snap toward vertical (noise influence drops to 0.1x) over 45 frames then gradually return to full displacement over 180 frames, creating visual punctuation. Background texture beziers slowly rotate (0.05 degrees/frame) around their midpoints. Highlight traces become more prominent (opacity increases from 30% to 65%) and flicker subtly using noise-driven opacity.

## Transformation
Individual strand transformation follows three-stage lifecycle. BIRTH (frames 0-120 of strand lifespan): Strand begins as straight vertical line, noise displacement gradually increases from 0 to full value using exponential ease-out (easeOutCubic). Opacity fades in from 0% to 100% over first 60 frames. MATURITY (frames 120-7200): Full noise displacement active, stroke weight oscillates between 1.5-3.5px using sin(frameCount * 0.003 + randomSeed) with unique phase offset per strand. Control points individually undergo micro-transformations: every 180-300 frames (randomized per point), a control point's noise influence multiplies by 1.6-2.1 for 90 frames creating localized bulge emphasis, then returns to baseline. DECAY (triggered randomly with 0.3% probability per frame after frame 1800): Noise influence begins declining linearly over 300 frames. Simultaneously, stroke weight thins from current value to 0.5px. Opacity fades to 0% over final 120 frames. Upon full decay, strand is reborn at random column position with new randomSeed values. Cluster nodes have independent lifecycle: EMERGENCE (45 frames): scale grows from 0 to 1.0 using ease-out-back (slight overshoot to 1.15, settle to 1.0). PERSISTENCE (variable, depends on strand proximity): scale pulses between 0.9-1.2 using perlin noise at 0.002 frequency. DISSOLUTION (30 frames): scale shrinks to 0 using ease-in-quad while opacity drops simultaneously.

## Rhythm
Macro rhythm is slow breathing cycle with 8-second inhale/exhale. Global noise time parameter follows sin(frameCount * 0.00131) * 0.5 + 0.5, creating gentle acceleration/deceleration of all undulation. This prevents metronomic feel. Micro rhythm: individual strand control points are offset by their y-position * 0.1 in noise space, creating cascading wave effect where movement ripples down strands at apparent 2-3px/frame velocity. Every 15-23 seconds (randomized), system undergoes brief 'tension spike' — all movement accelerates 2.5x for 2 seconds (120 frames) then smoothly returns to baseline over 3 seconds using exponential decay. Cluster node appearance/disappearance creates staccato visual rhythm: approximately 8-15 nodes visible at any moment, each with 1.5-2.5s lifespan, producing flickering connectivity pattern. Phase transitions are gradual: 3-second crossfades between phases using smooth step interpolation, avoiding jarring changes.

## Interaction (mouse)
Mouse X position (mapped 0-width to 0-1) controls global noise frequency multiplier: at left edge (mouseX=0), frequency is 0.5x normal creating slower, more languid undulation. At right edge (mouseX=width), frequency is 2.2x normal creating rapid, nervous oscillation. Mapping uses exponential curve: pow(mouseX/width, 1.6) for more dramatic response in right half. Mouse Y position (mapped 0-height to 0.3-1.2) controls strand tension parameter: at top (mouseY=0), maximum lateral displacement is reduced to 30% creating tighter, more constrained forms. At bottom (mouseY=height), displacement increases to 120% creating extreme bulging. Click spawns temporary 'disturbance wave': a radial force emanates from click position at 8px/frame, affecting all control points within 200px radius. Points are pushed away from click origin with force inversely proportional to distance (closer = stronger push, max 15px displacement). Wave persists for 180 frames, force decaying exponentially. During wave, affected strands increase stroke weight by 40% and reduce opacity to 70% creating visual 'shock'. Multiple clicks create overlapping waves. When mouse is stationary for >3 seconds, system gradually returns to autonomous baseline behavior with 4-second ease-out transition. Fallback when no mouse detected: autonomous mode uses noise(frameCount * 0.0003, frameCount * 0.0002) to simulate slow mouse movement across canvas, creating gentle parameter drift without user input.
