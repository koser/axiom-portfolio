# Serpentine Pursuit in Four Variations
**Date:** 2026-03-16
**Project ID:** AG19
**Library:** p5.js

## Concept
A 2x2 grid dividing the canvas into four distinct territories, each containing a 5x5 arrangement of pulsing circles (25 circles per quadrant, 100 total). A continuous black stroke inhabits each quadrant with fundamentally different wrapping logic: top-left uses tight spiral coiling around sequential circles, top-right employs figure-eight infinity patterns between circle pairs, bottom-left traces concentric orbital rings around circle clusters, bottom-right weaves angular zigzag paths connecting diagonal circle sequences. Each stroke has fixed length (1200px) and continuously grows from its head while retracting from its tail, creating perpetual forward motion. The strokes exhibit intelligent pathfinding behavior, steering toward the nearest unwrapped circle using bezier control points that curve around obstacles. Circles within each quadrant pulse in synchronized waves that propagate from the stroke's current position, creating ripple effects. As circles pulse, they emit subtle force fields (attraction radius 80px, strength 0.3) that pull the stroke's heading vector toward their centers. The stroke never stops evolving — when it completes wrapping a circle, it immediately seeks the next target, creating an endless dance of pursuit and embrace. Circles that have been recently wrapped glow briefly (2.4s fade) before returning to dormant state. The piece explores autonomous behavior, emergent pathfinding, and the beauty of constraint-based motion where intelligence arises from simple rules.

## Palette
#f7f7f2 (canvas background - warm paper white), #0d0d0d (primary stroke - ink black), #c9c9c4 (dormant circles - taupe gray), #050505 (stroke depth layer - void black with 40% opacity), #e2e2dc (circle pulse peak - warm highlight), #8a8a85 (circle force field visualization during debug - olive gray, 15% opacity)

## Composition
Canvas divided into precise 2x2 grid with 8px gap between quadrants creating visual separation. Each quadrant is 50% canvas width minus 4px, 50% canvas height minus 4px. Within each quadrant, circles arranged in perfect 5x5 grid with 18% quadrant-width spacing horizontally and vertically, circles sized at 7% of quadrant width (approximately 42px diameter on 1200px canvas). Top-left quadrant: stroke begins at grid position [0,0] moving clockwise in tight spiral. Top-right: stroke begins at [2,2] center, traces infinity loops. Bottom-left: stroke begins at [4,4] corner, moves in concentric rings inward. Bottom-right: stroke begins at [0,4], zigzags diagonally upward. Strokes never cross quadrant boundaries. Visual weight concentrated at active wrapping sites where stroke density is highest.

## Elements
Primary element: four independent continuous bezier strokes, each constructed from 180 connected segments (segment length 6.67px × 180 = 1200px total). Stroke width 3.8px with round caps and joins, rendered with depth using two-layer technique: shadow layer offset -1px/-1px at 40% opacity (#050505), top layer at full opacity (#0d0d0d). Each segment defined by position vector, heading angle, and two bezier control points offset perpendicular to heading by ±22px. Stroke head advances 2.4px per frame (40fps = 96px/second growth rate), tail retracts at identical rate maintaining fixed length. Secondary elements: 100 circles (25 per quadrant), each a data object storing position, base radius (21px), current pulse scale (1.0-1.35 multiplicative), pulse phase (0-2π), wrapped state (boolean), last-wrapped timestamp, force field strength. Circles rendered as filled shapes with 1.2px stroke outline. Tertiary elements: force field visualizations (debug mode only) shown as concentric rings at 80px radius with radial gradient falloff. Pathfinding targets rendered as small cross-hairs (+) at 8px size marking next destination for each stroke head.

## Animation
PHASE 1 (0-45s): INITIAL PURSUIT. All four strokes begin growth from their respective starting positions at 2.4px/frame. Top-left stroke coils clockwise around first circle in 7.2s (spiral radius contracts from 50px to 24px over 420 frames), then seeks next circle in sequence. Top-right stroke traces first figure-eight loop in 9.6s (two connected 4.8s loops, each 180° arc with 65px radius). Bottom-left begins outermost concentric ring at 90px radius, completes first orbit in 12.3s. Bottom-right zigzags toward [1,3] position in 6.1s using 38° angle changes every 180px. Circles pulse at base frequency 0.08Hz (12.5s period), amplitude 0.15 scale multiplier. Pulse waves propagate from stroke heads at 180px/s speed, triggering circles in 80px radius to synchronize phases. PHASE 2 (45-120s): ATTRACTION DYNAMICS. Force fields activate — circles within 80px of stroke heads emit attraction vectors that bias heading angle by up to 8° per frame toward circle center. Strokes exhibit smooth orbital insertion when approaching circles: at 45px distance, stroke begins 240° wrap maneuver over 3.2s, control points sweep through [current_angle - 30°, current_angle + 30°] creating smooth embrace. Pulse frequency increases to 0.12Hz when circles are being wrapped, creating visual excitement. Wrapped circles glow by lerping fill color toward #e2e2dc over 0.4s, then fade back over 2.0s exponential decay. Stroke pathfinding updates every 18 frames: raycast from head position in 16 directions, select nearest unwrapped circle not blocked by wrapped circles, set as target, interpolate heading toward target at 3.2°/frame. PHASE 3 (120-180s): EMERGENT COMPLEXITY. All circles have been wrapped at least once. Strokes now target least-recently-wrapped circles, creating revisitation patterns. Pulse synchronization becomes global within quadrants: all 25 circles in a quadrant phase-lock when stroke completes full circuit (detected when all circles wrapped within 18s window). Phase-locked quadrants pulse in unison at 0.15Hz with 0.25 amplitude. Stroke wrapping tightens: wrap radius decreases from 240° to 200° over phase 3, creating denser coiling. Control point offsets reduce from ±22px to ±16px, making curves tighter. Cross-quadrant visual rhythm emerges as different wrapping strategies (spiral vs infinity vs concentric vs zigzag) reach completion at different intervals, creating syncopated pattern where one quadrant is always in active wrap while others are in transit.

## Transformation
Stroke geometry transforms through bezier control point manipulation. Each segment's two control points (cp1, cp2) start perpendicular to heading vector at ±22px offset. As stroke approaches target circle (distance < 60px), control points smoothly rotate through arc: cp1 sweeps from [heading - 90°] to [heading + 30°] over approach time (1.8s), cp2 sweeps from [heading + 90°] to [heading - 30°], creating anticipatory curve that wraps around obstacle. During wrapping phase (3.2s duration), control points orbit around circle center maintaining 24px offset from circle edge, angles advance at 75°/second creating smooth encirclement. Post-wrap, control points snap back to perpendicular alignment over 0.6s using easeOutQuad interpolation. Circle transformations are scale-based: pulse scale oscillates using sine wave (scale = 1.0 + amplitude × sin(phase)), amplitude modulates based on proximity to stroke (near: 0.35, far: 0.15), phase advances at frequency rate × 2π per second. Wrapped circles additionally scale up by 1.12× for 0.4s then decay to 1.0× over 2.0s using exponential function (scale = 1.0 + 0.12 × e^(-t/0.8)). Stroke heading angle transforms through weighted vector sum: current_heading × 0.88 + target_direction × 0.08 + attraction_vector × 0.04, creating smooth steering that balances momentum, intent, and environmental influence.

## Rhythm
Micro-rhythm: stroke growth at steady 2.4px/frame (96px/s) creates constant forward drive, tail retraction at equal rate produces visual equilibrium. Circle pulse at 0.08-0.15Hz provides slow breathing tempo (6.67-12.5s periods). Wrapping events occur every 4-8s depending on inter-circle distance and pathfinding efficiency, creating irregular but frequent beats. Meso-rhythm: each complete circuit of a quadrant's circles takes 85-120s depending on wrapping strategy — spiral completes fastest (85s), zigzag slowest (120s) due to longer transit paths. Phase-lock events (all 25 circles synchronized) occur every 60-90s, marked by visual crescendo where entire quadrant pulses as one organism. Macro-rhythm: 180-second composition exhibits three-act structure — initial exploration (0-45s, strokes learn their territories), dynamic pursuit (45-120s, attraction forces dominate), and emergent order (120-180s, global synchronization). The piece has no true end state; at 180s it loops by resetting all wrapped-states but preserving stroke positions and circle phases, creating seamless continuation where strokes immediately pivot to newly-available targets. Cross-quadrant polyrhythm emerges from four independent wrapping cycles beating against each other — when top-left completes circuit in 85s, top-right is only 70% complete, creating phase relationships that shift throughout extended viewing.

## Interaction (passive)
No user interaction. The autonomous behavior system and pathfinding algorithms generate all motion. Strokes operate as independent agents with perception (circle detection within 200px forward cone), decision (target selection via raycast), and action (heading adjustment via weighted vector sum). Deterministic but appears organic due to complex feedback loops between attraction forces, pulse synchronization, and pathfinding choices.
