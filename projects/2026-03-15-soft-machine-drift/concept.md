# Soft Machine Drift
**Date:** 2026-03-15
**Library:** p5.js

## Concept
A field of biomorphic capsule-forms emerge and dissolve in overlapping clusters, their soft pill-shaped bodies constructed from bezier curves with noise-displaced edges. The forms breathe, rotate slowly, and occasionally split or merge, creating a tactile landscape of translucent color masses that recall pharmaceutical tablets, organic tissue, and modernist textile patterns. The piece explores tension between geometric intention and organic distortion.

## Palette
#f4f1e8 (pale cream background — quiet substrate), #4a7c8e (dusty teal — primary mass), #d94e72 (warm pink — secondary mass), #8b5a7d (muted plum — tertiary mass), #c9a84e (ochre yellow — rare accent), #2d5f3f (deep forest green — shadow layer), #e87451 (coral orange — punctuation)

## Composition
The canvas is divided into an implied 2x2 grid, though forms ignore these boundaries. Phase 1: clusters concentrate in upper-left and lower-right quadrants, leaving diagonal negative space. Phase 2: forms migrate toward center mass, creating dense overlapping stack with 60-70% opacity blending. Phase 3: the mass fractures outward, forms drift to canvas edges, corners become dense while center empties. Throughout, forms maintain 40-120px separation except during intentional overlap moments. Z-layering: 3 depth planes with different rotation speeds.

## Elements
PRIMARY: Capsule-bodies constructed via beginShape()/vertex()/bezierVertex()/endShape(CLOSE). Each capsule has 2 semicircular ends connected by parallel bezier curves. Base dimensions 80-180px length, 40-90px width. Edge displacement: Perlin noise sampled at 20-30 points around perimeter, offset 3-12px perpendicular to surface normal, noise frequency 0.08, evolves over time at 0.001/frame. Count: 24-32 capsules. SECONDARY: Trail ghosts — previous positions of capsules rendered at 8% opacity, creates motion blur echo. Each capsule leaves 8-15 ghost frames. TERTIARY: Micro-granules — tiny bezier blobs (6-12px) that cluster at capsule intersections, count 120-200, constructed from 6-vertex irregular polygons with corner rounding via quadraticVertex(). TEXTURE LAYER: Background noise texture from 4000 overlapping translucent ellipses (2-8px) at 3% opacity, regenerates every 480 frames.

## Animation
PHASE 1 (0-20s / 0-1200 frames): Capsules emerge from canvas edges at staggered intervals (1 every 35-50 frames). Each enters with initial velocity 0.8-1.5px/frame on vector pointing toward assigned grid quadrant, sinusoidal easing (sin(t * 0.05) * baseSpeed). Rotation: 0.003-0.008 radians/frame per capsule, each with unique angular velocity. Micro-granules spawn at 3/frame, drift randomly with Brownian motion (0.4px/frame velocity, direction changes every 12 frames). PHASE 2 (20-65s / 1200-3900 frames): Migration toward center begins at frame 1200. Each capsule's target position shifts to random point within 280px radius of canvas center. Movement: exponential easing (easeOutExpo with factor 0.08), speed increases to 2.1-3.2px/frame during first 8 seconds of phase, then decelerates to 0.3px/frame as targets approached. Rotation accelerates to 0.015 radians/frame. Overlap behavior: when two capsules' centers within 60px, they begin color blending over 180 frames (lerp between their base colors). Micro-granules now attract to capsule edges, forming halos. PHASE 3 (65s-120s / 3900-7200 frames): Dispersal. Each capsule receives new target at canvas periphery (within 80px of edge). Movement: cubic easing (easeInOutCubic), velocity peaks at 4.1px/frame at phase midpoint. Capsules begin splitting behavior: 30% chance per capsule to divide into two smaller capsules (60% original size) at frame intervals 4200, 4800, 5400. Split capsules inherit parent velocity + perpendicular offset (±1.8px/frame). Rotation decelerates to 0.004 radians/frame. Micro-granules dissipate, opacity fades to 0 over 600 frames. Trail ghosts lengthen to 22 frames. At frame 6800, cycle prepares to restart with fadeout.

## Transformation
CAPSULE LIFECYCLE: Birth (frames 0-90): scale from 0 to 1.0 using easeOutElastic (overshoot to 1.15 at frame 60, settle to 1.0). Opacity 0 to 0.75 linear. Edge noise displacement starts at 0, reaches full 3-12px range by frame 90. BREATHING: All capsules pulse using synchronized sin(frameCount * 0.02) * 0.08 + 1.0 scale multiplier, creates subtle 5-second breathing cycle. NOISE EVOLUTION: Edge displacement noise offset increments by 0.001/frame, causing slow morphing of capsule silhouettes — full shape transformation cycle is 1000 frames (16.7 seconds). COLOR SHIFT: Each capsule's hue shifts ±8° over 800 frame cycles using triangle wave function. SPLIT TRANSFORMATION (Phase 3): Parent capsule scale 1.0 to 0 over 45 frames (exponential), simultaneously two children emerge from parent center, scaling 0 to 0.6 over 60 frames with 40-frame delay offset between them.

## Rhythm
MACRO: Slow continental drift feel. Phase transitions are gradual (400-frame crossfades), not abrupt. Overall arc: gather (20s) → compress (45s) → scatter (55s) = 120-second complete cycle before restart. MICRO: Individual capsule movement has 5-second breathing rhythm overlaid on directional drift. Rotation is continuous but feels meditative (full rotation every 523-1047 frames). Micro-granules provide high-frequency jitter (position updates every frame) against capsules' slow glide. Trail ghosts create temporal smear, doubling perceived motion smoothness. Split events in Phase 3 occur in staggered bursts (3-5 splits within 200 frames, then 600-frame pause), creating punctuation in otherwise gradual dispersal.

## Interaction (mouse)
MOUSE POSITION: Mouse X (0 to width) controls global rotation speed multiplier: left edge = 0.5x speed, center = 1.0x, right edge = 2.2x speed, mapped with easeInOutQuad. Mouse Y (0 to height) controls edge noise displacement intensity: top = 50% displacement (1.5-6px range), bottom = 150% displacement (4.5-18px range), linear mapping. MOUSE PRESS: Click spawns attraction point at mouse coordinates lasting 240 frames. All capsules within 350px radius experience additional velocity toward attraction point at 1.8px/frame, using inverse square falloff (force = 1.8 * (1 - dist/350)^2). Attraction point renders as expanding ring of micro-granules (60 spawned on click, expand radially at 2.3px/frame, fade over 240 frames). PASSIVE FALLBACK: When mouse hasn't moved for 120 frames, system reverts to autonomous mode — rotation speed locked at 1.0x, noise displacement at 100% (3-12px), no attraction points.
