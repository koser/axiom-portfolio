# Sediment Collapse Liturgy
**Date:** 2026-03-15
**Library:** p5.js

## Concept
Stacked elliptical forms — representing geological strata or ritual cairns — slowly distort, fracture, and reassemble through noise-driven vertex displacement and gravitational collapse. The piece explores the tension between stable monumentality and gradual erosion, referencing the moodboard's serene stacked ovoids while introducing temporal decay and reformation cycles. The atmosphere is meditative yet unsettling — quiet disintegration punctuated by sudden reconstitution.

## Palette
#0d0d0f (void background — absolute darkness), #f4f1ed (primary mass — bone-white structural forms), #dad5cd (secondary — weathered shadow tones), #1a1a1c (tertiary — subtle edge darkening for depth)

## Composition
Seven horizontal strata begin dispersed across full canvas height (layers positioned at 15%, 25%, 35%, 50%, 65%, 80%, 92% of canvas height, widths 180-380px, heights 35-95px, horizontally centered). Over 120 seconds, the composition passes through three spatial states: Phase 1 maintains dispersed vertical distribution with orbital rotation around individual anchor points. Phase 2 triggers centripetal compression — all layers spiral inward toward canvas center (width/2, height/2) while shrinking and accelerating rotation. Phase 3 executes explosive radial expansion — layers blast outward to canvas edges (top/bottom 8% margins, left/right 5% margins) while rotating into horizontal alignment, then slowly drift back to dispersed origin state. Empty space migrates from distributed void to concentrated center, then explodes outward before reconsolidating.

## Elements
Each stratum is a custom closed shape built with curveVertex() creating smooth elliptical forms with 28 control points. Baseline dimensions: bottommost layer 380px wide × 95px tall, topmost 180px × 35px, graduated scaling between (layer n width = 380 - n×28.5px, height = 95 - n×8.5px). Visual character: edges are NOT clean curves — each vertex position is displaced by multi-octave Perlin noise (offset range ±4-14px Phase 1, ±8-28px Phase 2, ±2-8px Phase 3, noise frequency 0.02-0.09, using unique seed per vertex calculated as vertex.index × 0.347 + layer.seed × 2.719). This creates organic wobble, suggesting geological texture and material fatigue. Layers have subtle internal grain: 16 semi-transparent (#f4f1ed at 10% opacity) smaller elliptical shapes (50-80% of parent size, specific sizes 90-304px wide × 17.5-76px tall per parent scale) drawn inside each stratum using same vertex-displacement technique with independent noise seeds (grain.seed = layer.seed + grain.index × 1.234), creating sedimentary texture. No basic primitives. All forms are vertex-constructed with noise modulation.

## Animation
PHASE 1 (0-35s, 2100 frames at 60fps): Orbital dispersion. Each layer rotates around its own anchor point (initial Y positions at 15%, 25%, 35%, 50%, 65%, 80%, 92% of canvas height, all X at width/2) in elliptical orbit paths. Orbit dimensions per layer: layer 0 radius 45px × 25px, layer 1 radius 55px × 30px, layer 2 radius 65px × 35px, layer 3 radius 80px × 45px, layer 4 radius 70px × 40px, layer 5 radius 60px × 32px, layer 6 radius 50px × 28px. Orbit speeds (angular velocity in radians/frame): [0.0052, -0.0061, 0.0047, -0.0058, 0.0054, -0.0049, 0.0063] for layers 0-6 (negative = counterclockwise, positive = clockwise), creating polyrhythmic orbital periods from 16.3s to 21.7s. Orbit position calculated as: posX = anchorX + cos(frameCount × angularVelocity + layer.seed) × radiusX, posY = anchorY + sin(frameCount × angularVelocity + layer.seed) × radiusY. Layer self-rotation (independent of orbit): rotation = frameCount × [0.0008, -0.0011, 0.0013, -0.0009, 0.0012, -0.0010, 0.0014] radians/frame for layers 0-6, completing full 360° every 131-196 seconds (visible as slow roll). Scale pulses: ×0.96 to ×1.02 over 5.2-second (312 frame) sine cycles with unique phase offsets per layer (offsets 0.0rad, 0.9rad, 1.8rad, 2.7rad, 3.6rad, 4.5rad, 5.4rad, formula: scale = 0.99 + sin(frameCount × 0.0201 + offset) × 0.03). Vertex noise displacement: amplitude 9px base (±4-14px range), frequency 0.02, noise offset increments by 0.0012px/frame (full cycle every 1091 frames = 18.2s). PHASE 2 (35-72s, frames 2100-4320): Centripetal spiral collapse. Over 18 seconds (1080 frames), all layers spiral inward toward canvas center (width/2, height/2) using logarithmic spiral path: spiralAngle = startAngle + easeInQuad(t) × 4π (2 full rotations), spiralRadius = startRadius × (1 - easeInCubic(t)), where t = (frameCount - 2100)/1080, startAngle = atan2(startY - height/2, startX - width/2), startRadius = dist(startX, startY, width/2, height/2). Final clustered positions all within 40px radius of canvas center. Scale shrinks: ×1.0 to ×0.35 using easeInCubic over 18s. Self-rotation accelerates: rotationSpeed = baseSpeed × (1 + easeInQuad(t) × 8) reaching 9× initial speed by frame 4320, layers spinning 3-5 full rotations during inward spiral. Vertex noise displacement amplitude increases to 18px base (±8-28px range), frequency increases to 0.09, noise offset increment increases to 0.0035px/frame (full cycle every 298 frames = 5.0s, creating visual vibration). Hold compressed clustered state at center for 19s (1140 frames, frames 4320-5460) with continued fast rotation and high-frequency noise. PHASE 3 (72-120s, frames 4320-7200): Explosive radial expansion and reconstitution. At frame 5460 (91s mark), explosive expansion triggers: all layers blast outward on radial vectors from center to canvas perimeter over 4 seconds (240 frames) using easeOutQuad (1 - pow(1-t, 2)). Target positions: layers 0-6 positioned at [bottom 8%, bottom 20%, top 20%, left 5% + top 50%, right 5% + top 50%, top 8%, center top 15%] creating distributed edge formation. During expansion: scale grows ×0.35 to ×1.15 using easeOutElastic (overshoot formula: pow(2,-10×t) × sin((t×10-0.75)×(2×PI)/3) + 1) over 4s, reaching 115% scale then settling to 100% by frame 5520 (92s). Rotation decelerates: rotationSpeed = fastSpeed × (1 - easeOutCubic(t)), layers coast to stop with final accumulated rotation angles ranging -12deg to +18deg (varies per layer based on spiral accumulation). Vertex noise calms: amplitude decreases to 5px base (±2-8px range) over 4s using easeOutQuad, frequency drops to 0.025, noise offset increment slows to 0.0006px/frame (full cycle every 1745 frames = 29.1s). Hold expanded perimeter state for 10s (600 frames, frames 5700-6300). At frame 6300 (105s mark), gentle drift begins: all layers lerp back to original Phase 1 dispersed positions (15%, 25%, 35%, 50%, 65%, 80%, 92% heights, centered X) over 12 seconds (720 frames) using easeInOutQuad. Rotation angles lerp to 0deg. Scale stabilizes at ×1.0. Hold Phase 1 dispersed state for 3s (180 frames), orbital motion resumes smoothly.

## Transformation
Each layer's lifecycle: vertex positions recalculate every frame using time-evolving noise coordinates (xoff = vertex.baseX × noiseFrequency + noiseTimeOffset, yoff = vertex.baseY × noiseFrequency + noiseTimeOffset, noiseTimeOffset increments per phase schedule: 0.0012/frame Phase 1, 0.0035/frame Phase 2, 0.0006/frame Phase 3). Individual vertices persist throughout (no birth/death), positions computed as: displacedX = baseX + cos(normalAngle) × noise(xoff, yoff) × amplitude, displacedY = baseY + sin(normalAngle) × noise(xoff, yoff) × amplitude, where normalAngle is perpendicular to ellipse tangent at that vertex. Internal grain ellipses inherit parent layer transform matrix (position, rotation, scale) but maintain independent noise seeds (grain.noiseTimeOffset = layer.noiseTimeOffset + grain.index × 0.891), creating internal movement parallax. Opacity constant: 100% for primary forms (#f4f1ed), 10% for grain shapes. During Phase 2 spiral compression: orbital motion transitions to spiral via angle calculation spiralAngle = currentOrbitAngle + easeInQuad(t) × 4π, radius collapses via spiralRadius = currentOrbitRadius × (1 - easeInCubic(t)). Scale compression applied as transform matrix scale factor, not vertex recalculation. During Phase 3 expansion: radial vectors calculated as expansionAngle = atan2(currentY - height/2, currentX - width/2) + random(-0.3, 0.3) (5-degree wobble per layer for organic spread), expansionDistance = lerp(40px, targetDistance, easeOutQuad(t)) where targetDistance varies per layer (range 380-520px to reach canvas edges from center). Elastic overshoot on scale only, positions use clean easeOutQuad.

## Rhythm
Macro rhythm: 120-second (7200 frame) full cycle divided into three acts with shifting energy. Phase 1 (35s) is adagio — slow elliptical orbits with periods 16.3-21.7s (non-synchronous, creating continuous motion across canvas), 18.2s noise cycles provide gentle surface tremor, 5.2s scale pulses add subtle breathing. Phase 2 (37s) is accelerando — 18s inward spiral trajectory with exponentially increasing rotation speed (1× to 9×), centripetal motion creates visual suction, 19s chaotic hold at center with 5.0s rapid noise cycles (visual boiling). Phase 3 (48s) is fortissimo → diminuendo — explosive 4s outward blast with elastic overshoot (50% of motion in first 2s, overshoot peak at 3.2s), 10s tense hold at perimeter with 29.1s ultra-slow noise (almost static), 12s gentle drift homeward with symmetrical ease-in-out, 3s quiet pause. Micro rhythm: polyrhythmic orbits in Phase 1 — seven layers with orbital periods [16.3s, 17.5s, 18.0s, 18.4s, 19.7s, 21.1s, 21.7s] and alternating clockwise/counterclockwise directions create complex interference patterns, nearest harmonic alignment at ~388s (outside single cycle, never synchronizes). Self-rotation operates at 0.5-0.7× frequency of orbit (continuous accumulating spin). Phase 2 front-loads centripetal acceleration: 70% of radius collapse in first 45% of duration via cubic easing. Phase 3 explosive expansion front-loaded: 80% displacement in first 40% of 4s blast, elastic overshoot adds 15% extra travel then rebounds over next 1.2s.

## Interaction (mouse)
Mouse X position controls vertex noise amplitude multiplier in real-time: mouseX mapped from 0 to width maps to amplitudeMultiplier 0.3 to 3.5 (formula: multiplier = map(mouseX, 0, width, 0.3, 3.5, true)). Applied as: effectiveAmplitude = baseAmplitude × multiplier. At left edge (mouseX=0px) amplitude ×0.3 yields nearly smooth edges (±1.2-4.2px in Phase 1). At right edge (mouseX=width) amplitude ×3.5 yields extreme distortion (±14-49px in Phase 1). Mouse Y position controls global time scale (animation speed): mouseY mapped from 0 to height maps to timeScale 0.25 to 2.0 (formula: scale = map(mouseY, 0, height, 0.25, 2.0, true)). Applied by modifying effective frameCount: effectiveFrameCount = frameCount × timeScale. At top (mouseY=0px) animation runs at 25% speed (extreme slow-motion). At bottom (mouseY=height) animation runs at 200% speed (double-time). This affects all orbital motion, spirals, noise evolution, and phase transitions proportionally. Mouse click (mousePressed event) triggers repulsion burst: duration 0.8s (48 frames). All layers receive outward impulse vector from click position: impulseAngle = atan2(layer.currentY - mouseY, layer.currentX - mouseX), impulseMagnitude = map(clickDistance, 0, 300, 180, 30, true) where clickDistance = dist(layer.currentX, layer.currentY, mouseX, mouseY), capped at 300px max sensing range. Layers within range receive positional offset: offsetX = cos(impulseAngle) × impulseMagnitude, offsetY = sin(impulseAngle) × impulseMagnitude, applied using easeOutCirc (formula: sqrt(1 - pow(t-1, 2))) over 48 frames. Simultaneously, clicked layers rotate by impulseRotation = random(-45, 45) degrees over 0.8s using easeOutQuad, then return to phase-appropriate rotation over next 1.2s (72 frames) using easeInOutQuad. Vertex noise spikes to 2.5× amplitude for clicked layers only during burst duration. Multiple rapid clicks create overlapping repulsion waves. Click event does not reset phase timer. Passive fallback: track lastMouseMoveTime using millis(), update on mouseMoved() callback. If (millis() - lastMouseMoveTime > 8000), set virtual mouseX = width × 0.45 and mouseY = height × 0.6, applying moderate distortion (amplitudeMultiplier = 1.5, timeScale = 1.1).
