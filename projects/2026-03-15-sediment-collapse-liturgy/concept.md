# Sediment Collapse Liturgy
**Date:** 2026-03-15
**Library:** p5.js

## Concept
Stacked elliptical forms — representing geological strata or ritual cairns — slowly distort, fracture, and reassemble through noise-driven vertex displacement and gravitational collapse. The piece explores the tension between stable monumentality and gradual erosion, referencing the moodboard's serene stacked ovoids while introducing temporal decay and reformation cycles. The atmosphere is meditative yet unsettling — quiet disintegration punctuated by sudden reconstitution.

## Palette
#0d0d0f (void background — absolute darkness), #f4f1ed (primary mass — bone-white structural forms), #dad5cd (secondary — weathered shadow tones), #1a1a1c (tertiary — subtle edge darkening for depth)

## Composition
Seven horizontal strata occupy the vertical center, initially arranged in perfect vertical alignment forming a cairn-like tower (width 180-380px per layer, heights 35-95px, centered at canvas midpoint). Over 120 seconds, the composition passes through three spatial states: Phase 1 maintains central vertical alignment with subtle breathing. Phase 2 introduces horizontal drift — upper layers slide left/right by 40-120px, breaking the tower's symmetry. Phase 3 triggers downward collapse — layers compress toward bottom third of canvas while rotating 3-15 degrees, creating a landslide formation. Empty space migrates from edges to center, then from top to bottom.

## Elements
Each stratum is a custom closed shape built with curveVertex() creating smooth elliptical forms with 28 control points. Baseline dimensions: bottommost layer 380px wide × 95px tall, topmost 180px × 35px, graduated scaling between (layer n width = 380 - n×28.5px, height = 95 - n×8.5px). Visual character: edges are NOT clean curves — each vertex position is displaced by multi-octave Perlin noise (offset range ±3-12px Phase 1, ±6-24px Phase 2, ±8-32px Phase 3, noise frequency 0.015-0.08, using unique seed per vertex calculated as vertex.index × 0.347 + layer.seed × 2.719). This creates organic wobble, suggesting geological texture and material fatigue. Layers have subtle internal grain: 16 semi-transparent (#f4f1ed at 10% opacity) smaller elliptical shapes (50-80% of parent size, specific sizes 90-304px wide × 17.5-76px tall per parent scale) drawn inside each stratum using same vertex-displacement technique with independent noise seeds (grain.seed = layer.seed + grain.index × 1.234), creating sedimentary texture. No basic primitives. All forms are vertex-constructed with noise modulation.

## Animation
PHASE 1 (0-30s, 1800 frames at 60fps): Breathing stasis. All seven layers remain vertically centered. Each layer oscillates scale: ×0.98 to ×1.04 over 4.8-second (288 frame) sinusoidal cycles (unique phase offset per layer: layer 0 offset 0.0rad, layer 1 offset 0.7rad, layer 2 offset 1.4rad, layer 3 offset 2.1rad, layer 4 offset 2.8rad, layer 5 offset 3.5rad, layer 6 offset 4.2rad, formula: scale = 1.01 + sin(frameCount × 0.0218 + offset) × 0.03). Vertex noise displacement animates: noise offset increments by 0.0008px/frame, creating slow surface drift (full cycle every 1308 frames = 21.8s at frequency 0.015). Vertical position wobble: ±1.5px using sin(frameCount × 0.02 + layer.seed) per layer. Zero rotation. Displacement amplitude: baseAmplitude = 7.5px (range ±3-12px per vertex using noise()). PHASE 2 (30-75s, frames 1800-4500): Lateral drift and intensifying deformation. Horizontal positions drift over 15 seconds (900 frames, elapsed 0-900 in phase): layer 0 (bottom) stays at centerX, layers 1-6 drift using cubic easing (t < 0.5 ? 4×t³ : 1-pow(-2×t+2,3)/2) to target offsets [-40px, +65px, -85px, +110px, -120px, +95px] from original centerX. Vertex noise displacement amplitude increases to 15px base (±6-24px range), frequency increases to 0.06. Noise offset increment increases to 0.002px/frame (full cycle every 524 frames = 8.7s). Scale oscillation continues with increased amplitude: ×0.95 to ×1.07 (formula: scale = 1.01 + sin(frameCount × 0.0218 + offset) × 0.06). Rotation begins for layers 4-6 (top 3): rotation = sin(frameCount × 0.00131 + layer.seed) × [2deg, 3deg, 4deg] respectively. PHASE 3 (75-120s, frames 4500-7200): Gravitational collapse (75-83s, frames 4500-4980) and reconstitution (110-116s, frames 6600-6960). Collapse phase (480 frames): all layers accelerate downward using easeInQuad (t×t) over 8 seconds to compress into bottom 40% of canvas — target Y positions: layer 0 at height×0.95, layer 1 at height×0.88, layer 2 at height×0.81, layer 3 at height×0.74, layer 4 at height×0.68, layer 5 at height×0.62, layer 6 at height×0.57 (gaps 25-60px). Rotation intensifies to final angles over 12 seconds (720 frames) using easeInQuad: [+3deg, -7deg, +12deg, -15deg, +9deg, -5deg, 0deg] for layers 0-6. Heights compress by 20% (×0.80), widths expand by 10% (×1.10) over collapse duration. Vertex displacement reaches maximum: 20px base amplitude (±8-32px range), noise frequency 0.08, increment 0.004px/frame (full cycle every 262 frames = 4.4s). Hold chaotic state for 1620 frames (27s). At frame 6600 (110s mark), reconstitution begins: all properties lerp back to Phase 1 values using easeInOutCubic over 6 seconds (360 frames) — positions return to vertical centerline, rotations to 0deg, dimensions to base values, noise calms to 7.5px amplitude and 0.015 frequency. Hold Phase 1 state for 10 seconds (600 frames), then cycle repeats.

## Transformation
Each layer's lifecycle: vertex positions recalculate every frame using time-evolving noise coordinates (xoff = vertex.baseX × noiseFrequency + noiseTimeOffset, yoff = vertex.baseY × noiseFrequency + noiseTimeOffset, noiseTimeOffset increments per phase schedule: 0.0008/frame Phase 1, 0.002/frame Phase 2, 0.004/frame Phase 3). Individual vertices persist throughout (no birth/death), positions computed as: displacedX = baseX + cos(normalAngle) × noise(xoff, yoff) × amplitude, displacedY = baseY + sin(normalAngle) × noise(xoff, yoff) × amplitude, where normalAngle is perpendicular to ellipse tangent at that vertex. Internal grain ellipses inherit parent layer transform matrix (position, rotation, scale) but maintain independent noise seeds (grain.noiseTimeOffset = layer.noiseTimeOffset + grain.index × 0.891), creating internal movement parallax. Opacity constant: 100% for primary forms (#f4f1ed), 10% for grain shapes. During collapse Phase 3 compression (frames 4500-4980): heights multiply by compressionFactor = lerp(1.0, 0.80, easeInQuad(t)) where t = (frameCount - 4500)/480, widths multiply by expansionFactor = lerp(1.0, 1.10, easeInQuad(t)). During reconstitution (frames 6600-6960): all deformations reverse using reverse interpolation to base values.

## Rhythm
Macro rhythm: 120-second (7200 frame) full cycle divided into three acts with accelerating tension. Phase 1 (30s) is largo — slow 4.8s breath cycles, 21.8s noise cycles. Phase 2 (45s) is andante moderato — 15s drift transition feels like gradual tilting, 8.7s noise cycles create increasing tremor. Phase 3 is allegro → subito piano — dramatic 8s (480 frame) collapse front-loaded via exponential easing (50% of motion in first 4s/240 frames), 27s chaotic hold with 4.4s noise cycles (intense vibration), then 6s (360 frame) smooth restoration via cubic easing with symmetrical acceleration/deceleration, 10s (600 frame) meditative pause. Micro rhythm: polyrhythmic breathing — seven layers oscillate on phase-staggered sine waves (offsets 0rad, 0.7rad, 1.4rad, 2.1rad, 2.8rad, 3.5rad, 4.2rad = intervals of 40.1 degrees), never synchronizing, creating rolling wave of expansion/contraction through the stack. Vertex wobble operates at 3-5× frequency of breathing (21.8s vs 4.8s in Phase 1) generating high-frequency tremor beneath slow undulation. Collapse motion front-loaded: 75% displacement in first 50% of duration. Reconstitution smooth and even: symmetrical ease-in and ease-out.

## Interaction (mouse)
Mouse X position controls noise frequency multiplier in real-time: mouseX mapped from 0 to width maps to frequencyMultiplier 0.5 to 2.5 (formula: multiplier = map(mouseX, 0, width, 0.5, 2.5, true)). Applied as: effectiveFrequency = baseFrequency × multiplier. At left edge (mouseX=0px) frequency ×0.5 yields smoother organic edges (0.0075 in Phase 1). At right edge (mouseX=width) frequency ×2.5 yields chaotic edges (0.0375 in Phase 1). Mouse Y position controls cumulative rotation force: mouseY mapped from 0 to height maps to rotationBoost 0deg to +25deg (formula: boost = map(mouseY, 0, height, 0, 25, true)). Applied additively to all phase-driven layer rotations (finalRotation = phaseRotation + rotationBoost). At top (mouseY=0px) zero boost. At bottom (mouseY=height) adds +25deg to all layers. Mouse click (mousePressed event) triggers immediate collapse event: duration 1.5s (90 frames) to compressed state, then 2.0s (120 frames) elastic return. Collapse: all layers lerp centerX/centerY to canvas center (width/2, height/2), each layer rotates by random(-90, 90) degrees (unique per layer, seeded by layer.index × millis()), vertex noise amplitude multiplies by 4× (spikes to ±12-48px Phase 1, ±24-96px Phase 2, ±32-128px Phase 3), using linear easing over 90 frames. Elastic return uses easeOutElastic (formula: pow(2,-10×t) × sin((t×10-0.75)×(2×PI)/3) + 1) over 120 frames to overshoot by 15% then settle back to phase-appropriate positions/rotations. Click event persists through phase transitions (doesn't reset phase timer). Passive fallback: track lastMouseMoveTime using millis(), update on mouseMoved() callback. If (millis() - lastMouseMoveTime > 8000), set virtual mouseX = width/2 and mouseY = height/3, applying neutral distortion (frequencyMultiplier = 1.5, rotationBoost = 8.33deg).
