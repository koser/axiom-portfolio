# Vertical Erosion Field
**Date:** 2026-03-16
**Project ID:** AG22
**Library:** p5.js

## Concept
A dense vertical stripe structure undergoes continuous erosion and reformation through hundreds of small geometric fragments that peel away from rigid bands and migrate across the canvas. The piece captures the moodboard's stark black-and-white contrast, sharp vertical rhythm, and the organic decay visible in the reference — transforming architectural rigidity into dynamic disintegration and rebirth.

## Palette
#0d0d0d (deep black background), #ffffff (primary — harsh vertical bands), #e8e8e8 (secondary — eroding fragments in mid-flight), #1a1a1a (accent — shadow zones between stripes)

## Composition
The canvas divides into 18-24 vertical bands of varying widths (40-80px) that span full height. Initial state (0-15s): bands are solid, crisp, evenly spaced with narrow black gaps. Transition state (15-45s): bands develop irregular edge erosion as small rectangular fragments (4x12px to 8x20px) break off and drift horizontally, creating a mid-ground layer of migrating debris. The left third of the canvas erodes faster, creating asymmetric density. Late state (45s+): bands begin rebuilding from accumulated fragments at screen edges, reversing the flow — fragments now migrate inward and reattach, restoring solidity in pulsing waves from left to right. Negative space (black gaps) breathes wider and narrower (±15px) on 8-second noise-driven cycles.

## Elements
PRIMARY — Vertical bands constructed as custom shapes using beginShape()/vertex() with noise-displaced edges: each band is a polygon with 60-80 vertices along each vertical edge, displaced by Perlin noise (noise scale 0.02, amplitude 2-8px varying per band). Bands are 40-80px wide, full canvas height, positioned at regular intervals but with slight horizontal drift (±5px per frame at 0.1px/frame). SECONDARY — Erosion fragments generated by sampling random points along band edges: each fragment is a small rectangle (4-20px wide, 8-24px tall) with one or two corners beveled using vertex() to create irregular chips. Fragments have rotation (0-45 degrees), opacity (0.6-0.95), and semi-transparent fill. 300-500 fragments active simultaneously. TERTIARY — Shadow trails behind fragments: each fragment leaves a 6-frame trail using overlapping rectangles at 70%, 50%, 30% opacity of parent, offset by velocity vector. Trails create directional blur effect mimicking motion photography.

## Animation
PHASE 1 (0-15s, 900 frames): Bands stand rigid with subtle edge noise animation (vertices update every frame with noise increment 0.01). Every 1.2 seconds, 15-25 new fragments spawn from random points on band edges. Fragments have initial horizontal velocity 0.8-2.5px/frame (positive for right-moving, negative for left-moving — 60% right, 40% left). Vertical velocity is minimal (-0.2 to +0.2px/frame). Fragments use cubic easing for first 90 frames (acceleration phase). Opacity starts at 0 and ramps to 0.85 over 30 frames (linear). PHASE 2 (15-45s, 900-2700 frames): Fragment spawn rate increases to 30-40 per 1.2s burst. Bands begin slow horizontal drift — left third drifts leftward at 0.15px/frame, right third drifts rightward at 0.1px/frame, center remains anchored. Edge noise amplitude increases to 6-12px. Fragments now have lifespan variation: 50% last 180 frames, 30% last 300 frames, 20% last 450 frames. When fragments reach 90% of their lifespan, opacity begins exponential decay to 0 over final 10%. Fragment rotation increases by 0.3-0.8 degrees/frame throughout lifetime. Gap width between bands oscillates sinusoidally (8-second period, ±15px amplitude). PHASE 3 (45s+, 2700+ frames): Flow reverses. Fragments now spawn from left/right screen edges (40% from left edge, 40% from right edge, 20% still from band edges). Edge-spawned fragments have velocity directed toward nearest band (2.0-3.5px/frame horizontal component, calculated using atan2 targeting). When fragment reaches within 5px of band edge, it merges — band edge vertices in contact zone shift outward by 2-4px over 60 frames (exponential easing), effectively growing the band. Bands pulse with scale variations (98%-102% of original width) on 4-second cycles per band, offset by 0.3s between adjacent bands for wave effect. Fragment spawn rate decreases to 10-15 per 1.2s. Every 12 seconds, a full horizontal sweep occurs: 80-120 fragments spawn simultaneously from one screen edge, creating a dense cloud that migrates across canvas at uniform 2.2px/frame speed with vertical spread (±50px from midpoint using Perlin noise), temporarily overwhelming the composition before dissipating.

## Transformation
Individual fragments undergo three-stage life: SPAWN (frames 0-30): scale from 0.3 to 1.0 (exponential easing out), opacity 0 to 0.85 (linear), rotation set at spawn angle. DRIFT (frames 30 to lifespan-30): constant velocity with slight Perlin noise perturbation (±0.3px/frame in both axes, noise scale 0.01), rotation increment 0.3-0.8 deg/frame, scale holds at 1.0, opacity stable. FADE (final 30 frames): velocity decays to 0 using exponential easing (multiplied by 0.92 per frame), opacity exponential decay to 0, scale shrinks to 0.5, rotation rate doubles for final spin. Bands transform through edge vertex displacement: each vertex has independent noise seed, updates position every frame based on noise(seed + frameCount * 0.01), displacement amplitude modulates with Phase (2-8px in Phase 1, 6-12px in Phase 2, 4-10px in Phase 3). During merge events (Phase 3), contacted vertices move outward 2-4px over 60 frames using easeOutExpo curve.

## Rhythm
MACRO: Three-phase structure with 15-second phase transitions creates large-scale temporal arc — opening solidity, middle chaos, closing reformation. 12-second sweep events in Phase 3 provide dramatic punctuation. Overall loop: 90 seconds, then Phase 3 behaviour continues indefinitely with increasing density. MICRO: Fragment spawn bursts every 1.2 seconds create staccato pulse. Individual fragment lifespans (180-450 frames = 3-7.5 seconds) create overlapping generations. Band gap oscillation (8-second period) provides slow breath beneath faster fragment activity. Band pulse waves (4-second period with 0.3s offset cascade) create traveling rhythm left-to-right every 7.2 seconds (24 bands × 0.3s). Trail frames (6-frame persistence) create 100ms motion smear, adding fluidity to otherwise discrete fragments.

## Interaction (mouse)
Mouse X position (mapped 0-width) controls fragment horizontal velocity bias: left half of screen increases leftward-moving fragment percentage (up to 80% left at x=0), right half increases rightward percentage (up to 80% right at x=width). Mouse Y position (mapped 0-height) controls fragment spawn rate: top quarter spawns 5-10 per burst, second quarter 15-25 per burst, third quarter 30-50 per burst, bottom quarter 60-90 per burst. Mouse click spawns immediate radial burst from click location: 40-60 fragments with initial velocity 3.5-7px/frame radiating outward (angle = random(TWO_PI)), lifespan 240 frames, rotation rate 1.2-2.0 deg/frame. Click fragments have higher opacity (0.95) and larger size (12-28px wide). When mouse is stationary for 3+ seconds, band edge noise amplitude gradually increases to 18px over 2 seconds (exponential ease), creating localized turbulence. Mouse movement resets this timer. Fallback for no mouse input: autonomous mode uses Perlin noise field (scale 0.003) to modulate spawn rate (10-40 per burst) and velocity bias (sampled from noise at canvas center), creating organic variation without user control.
