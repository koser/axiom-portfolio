# Void Divisions
**Date:** 2026-03-15
**Library:** p5.js

## Concept
A generative study of organic cellular division constrained by rigid geometry. Black voids pulse and subdivide within white ovoid chambers, creating a breathing tension between soft biological forms and hard mathematical boundaries. The piece explores containment, growth, and the moment before symmetry breaks into chaos — directly translating the moodboard's stark monochrome language and curved-edge-meeting-straight-edge visual tension.

## Palette
#0d0d0d (deep void — the hungry black that consumes space), #f4f4f0 (bone white — the constraining membrane), #1a1a1a (shadow black — transitional darkness), #e8e8e4 (off-white — secondary membrane for depth layering)

## Composition
The viewport is divided into an invisible 3x3 grid. Center mass (middle cell) remains the gravitational anchor throughout. Phase 1: single large organic void occupies 60% of center space, contained by a thick white circular boundary (stroke weight 18px, diameter 420px). Phase 2: the boundary fractures into 3 overlapping elliptical containers (major axis 380-520px, minor axis 280-440px), each rotated at 0°, 120°, 240°, creating a triadic Venn structure. Voids subdivide to fill each chamber. Phase 3: containers migrate toward edges — one to upper-left quadrant, one to lower-right, one to right-center. Empty space becomes the dominant compositional force. Each container carries 2-4 void subdivisions. The center empties completely by 45 seconds, creating a breathing hollow.

## Elements
PRIMARY: Void Bodies — constructed using curveVertex() with 16-24 anchor points, each displaced by noise(xoff, yoff, zoff) where zoff increments by 0.003/frame, creating slow organic wobble. Size range: 80px to 340px equivalent diameter. Edge softness achieved through 3 concentric layers at 100%, 60%, 30% opacity, each offset by 2-4px of noise displacement. Count: starts at 1, subdivides to maximum 9 by phase 3. Visual character: metaball-like soft bodies with breathing edges, not perfect circles — think cell membranes under tension. SECONDARY: Container Rings — drawn using overlapping bezierVertex() curves (8 curve segments per ring) to create imperfect ellipses with thickness variation (14-22px stroke weight modulated by sin(frameCount * 0.04)). The rings aren't stroked ellipses but constructed from filled bezier shapes creating a thick band with inner and outer edge wobble (noise displacement on both boundaries, different noise seeds). TERTIARY: Connection Threads — when void bodies get within 60px of each other, spawn bezier curves between nearest edge points. Each thread: 4-7 control points with perpendicular noise offset (amplitude 8-18px), stroke weight 1.5-3.5px fading with distance, lifespan 180-420 frames with ease-out opacity decay. These create temporary cellular bridges. TEXTURE: Background noise field using overlapping semi-transparent bezier blobs (200-400 blobs, size 3-12px, opacity 1-4%, gray values #0d0d0d to #2a2a2a) regenerated every 420 frames to create subtle material grain.

## Animation
PHASE 1 (0-12s, frames 0-720): Single central void body breathes from 180px to 340px diameter following sin(frameCount * 0.008) * 80 + 260. Container ring also breathes inversely (420px to 480px diameter) creating compression/expansion rhythm. Void edge noise scrolls at zoff += 0.003/frame. Container ring thickness oscillates 14px to 22px at 0.04 cycle/frame. No subdivision yet. Movement: void drifts in tight circle (radius 15px, rotation speed 0.6°/frame). PHASE 2 (12-35s, frames 720-2100): Container ring splits into 3 overlapping ellipses over 180 frames using exponential easing (easeInOutCubic). Each ellipse rotates to final position at 1.2°/frame. At frame 900, first void subdivision triggers: central void scales down to 0.6x size over 30 frames (exponential ease-in), simultaneously spawns 3 child voids at 120° intervals, radius 90px from parent center, child size 0.4x parent. All voids now drift independently using Perlin noise flow field (angle = noise(x*0.004, y*0.004, frameCount*0.002) * TWO_PI * 1.8, speed 0.8-1.6px/frame). Connection threads begin spawning when voids approach each other. Second subdivision at frame 1500: each of the 3 voids splits into 2 (total 6 voids), smaller sizes (60-120px range). PHASE 3 (35s-70s, frames 2100-4200): Container ellipses migrate to edge positions over 600 frames (sinusoidal easing). Target positions: ellipse_1 center at (width*0.25, height*0.25), ellipse_2 at (width*0.82, height*0.5), ellipse_3 at (width*0.7, height*0.78). Voids follow their container assignments (2-3 voids per container) but with 120-frame lag, creating delayed trailing motion. Flow field becomes more turbulent (noise scale increases to 0.007, angle multiplier to 2.4). Final subdivision at frame 3000: 3 smallest voids each spawn 1 child (total 9 voids), sizes 40-80px. Center void completely dissolves over 240 frames (opacity 100% to 0%, size shrinks to 0.2x, exponential ease-in). Background texture regenerates every 420 frames with 60-frame crossfade. At frame 3900, all voids begin slow scale pulsing (0.85x to 1.15x, cycle 4.2 seconds, phase offset by void index * 0.4 radians).

## Transformation
Individual void lifecycle: BIRTH (frames 0-90): scale from 0x to 1x using easeOutElastic (overshoot 1.2x at frame 60, settle at 1x by frame 90), opacity 0% to 100% linear over first 30 frames, edge noise amplitude starts at 0px, ramps to full 4-8px displacement by frame 90. MATURITY (90+ frames): continuous edge wobble via noise, breathing scale modulation ±8-12% following slow sin(frameCount * 0.004 + unique_phase_offset), rotation 0.1-0.3°/frame. SUBDIVISION (triggered event): parent shrinks to 0.6x over 30 frames (exponential), opacity drops to 75%, spawns 2-3 children at polar offsets (radius = parent_size * 0.8, angles evenly distributed), children inherit parent velocity + radial kick (2-4px/frame outward, decays over 120 frames). DEATH (triggered at phase 3 center void, or when void exits container by >100px): scale shrinks exponentially to 0.1x over 240 frames, opacity fades 100% to 0% over same duration, edge noise amplitude increases 3x creating dissolution effect, connection threads detach and fade individually over final 60 frames. Container rings transform continuously: thickness modulation via sin(frameCount * 0.04 + ring_id * 1.2) * 4 + 18, inner/outer edge noise scrolls at different rates (outer zoff += 0.003, inner zoff += 0.0045), creating breathing organic boundary. During phase 2 split, original ring alpha fades 100% to 0% over 180 frames while 3 new rings fade in 0% to 100% over same period with 30-frame offset.

## Rhythm
Macro rhythm: breath cycle at 8.4-second intervals (frameCount % 504 == 0 triggers global synchronization event — all voids briefly align scale phase). The piece moves through compression (0-12s: everything tightening to center), explosion (12-35s: subdivision and distribution), and dissipation (35-70s: migration to edges, center emptying). Micro rhythm: void edge noise creates 3-second flutter cycles (frequency 0.003/frame = ~5 frames per wobble at 60fps). Container thickness pulses at 2.5-second intervals (0.04 cycle/frame). Connection thread spawning is stochastic but rate-limited: check every 6 frames (10 checks/second), spawn probability 35% when conditions met, creating irregular threading bursts. Subdivision events are deterministic shocks: frame 900 (15s), frame 1500 (25s), frame 3000 (50s) — each triggers 180-frame ripple where all voids briefly accelerate (velocity * 1.6 for 30 frames, exponential decay back to 1x over remaining 150 frames). Background texture regeneration every 7 seconds (420 frames) with 1-second crossfade creates subtle environmental pulse. Overall pacing: slow->medium->slower, with punctuated subdivision accelerations. The final 20 seconds (frames 3000-4200) slow to near-stillness as voids settle into edge orbits around their containers, only edge wobble and scale breathing remain active.

## Interaction (passive)
Autonomous driver system uses three layered noise fields: FIELD_A (global drift) with noise(x*0.004, y*0.004, frameCount*0.002) influences all void positions with 0.8-1.6px/frame velocity, updates every frame. FIELD_B (container positions) uses noise(container_id*0.8, frameCount*0.0008) to add micro-jitter (±1-3px/frame) to container target positions during phase 3 migration, creates natural settling rather than rigid stops. FIELD_C (subdivision timing variance) checks noise(frameCount*0.001, void_id*1.3) against threshold 0.55 to introduce ±60 frame variance in subdivision trigger times (so frame 900 event actually triggers between frames 840-960 per void). Pseudo-random seed reset every 4200 frames (70 seconds) ensures piece loops with variation — same general structure, different specific positions/timings. When piece reaches frame 4200, all voids fade out over 180 frames (exponential opacity decay), containers dissolve over 240 frames (stroke weight shrinks to 0, opacity to 0%), frame 4440 triggers hard reset to frame 0 with new noise seed, creating 70-second looping experience with 6-second transition/rebirth gap. No user input required or processed.
