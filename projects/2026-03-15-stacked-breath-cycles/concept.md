# Stacked Breath Cycles
**Date:** 2026-03-15
**Library:** p5.js

## Concept
A study in horizontal compression and release inspired by the moodboard's stacked elliptical forms. Seven undulating strands of custom bezier-constructed masses expand and contract in offset breathing patterns, creating hypnotic waves of negative space. The piece explores the tension between order and organic distortion through asymmetric wobble and phase-shifted rhythms.

## Palette
#e8e4db (warm neutral background — the moodboard's soft beige field), #1a1816 (primary — the deep black masses), #2e2a26 (secondary — softer shadow strands for depth), #b8b0a4 (accent — rare highlight glow on strand edges during exhale moments)

## Composition
Seven horizontal strands arranged in vertical stack formation across the center third of the canvas (from y: 25% to y: 75%). Each strand occupies approximately 8% of canvas height with 2% negative space between. The top and bottom strands have anchor dots (circular terminals with 12-18px diameter, noise-displaced edges). Composition shifts occur through horizontal stretch: Phase 1 keeps strands centered with symmetrical breathing. Phase 2 introduces lateral drift — odd-numbered strands shift left (0-60px over 15s), even-numbered shift right. Phase 3 returns to center with accelerated breathing creating vertical oscillation (±15px) in strand positions.

## Elements
Seven custom horizontal strands, each constructed using bezierVertex() to create elliptical masses with bulbous terminals. Construction: each strand is a closed bezier shape with 8-12 control points, width ranging 180-280px, height 45-65px. Edges are displaced using 2D Perlin noise (noiseDetail(3, 0.6), frequency 0.015) creating subtle wobble that updates each frame. The noise displacement range is ±8px perpendicular to the strand's longitudinal axis. Terminal dots (top strand, bottom strand) are constructed as vertex-based circles with 24 segments, each vertex displaced by noise (frequency 0.02, amplitude ±3px) creating breathing organic edges. Strand fill is solid primary color with 2px stroke weight using secondary color, applied only during contraction phase (opacity fade in over 20 frames, fade out over 30 frames). A subtle gradient overlay is achieved by drawing each strand twice: once at full opacity, once at 15% opacity offset by 2px downward to suggest dimensionality.

## Animation
Phase 1 (0-12s): All seven strands breathe in synchronized pairs. Strands 1&7 share a cycle (4.2s inhale using easeInOutCubic, 3.8s exhale using easeOutQuad). Strands 2&6 are offset by 1.2s (start at 30% into the cycle). Strands 3&5 offset by 2.4s. Strand 4 (center) is solo, 5s cycle with linear easing. Horizontal scale oscillates from 1.0 to 1.35, vertical scale from 1.0 to 0.88 (compression effect). Noise offset for edge wobble advances at 0.002 units/frame creating slow drift. Phase 2 (12-35s): Breathing continues but lateral drift introduced. Odd strands (1,3,5,7) translate left using sine wave motion (amplitude 60px, period 18s, easeInOutSine). Even strands (2,4,6) drift right with same parameters but inverted phase. Breathing cycles gradually accelerate — cycle time reduces from 4.2s to 2.8s over 23s duration using exponential interpolation (factor 0.985 per second). Stroke weight appears on contraction (fade in over 20 frames to 2px, fade out over 30 frames to 0px). Phase 3 (35s+): Lateral drift reverses (easeOutQuart return to center over 8s). Breathing now modulates vertical position — strands oscillate ±15px on y-axis synchronized with their scale breathing (when horizontally compressed, strands rise; when extended, they sink). Period is 3.2s with easeInOutSine. Terminal dots pulse scale independently: 1.0 to 1.4 over 2.1s cycles, offset from strand breathing by 0.7s. Every 12s, a single random strand exhibits a 'catch' — pauses at peak compression for 0.8s (hold at scale 1.35, 0.88) before rapid exhale over 0.6s.

## Transformation
Each strand transforms through continuous scale modulation. Horizontal scale range: 1.0 to 1.35 (35% expansion). Vertical scale range: 1.0 to 0.88 (12% compression). Transformation uses bezier easing curves — inhale is easeInOutCubic (gradual acceleration, gradual deceleration), exhale is easeOutQuad (fast release, slow settle). During the 'catch' event (Phase 3), the affected strand holds scale for 48 frames then snaps to 0.95 horizontal scale over 12 frames (overshoot) before rebounding to 1.0 over 24 frames (easeOutElastic with 1 bounce). Noise displacement on edges continuously evolves — noise offset advances 0.002/frame in x-direction, 0.001/frame in y-direction, creating slow morphing wobble that never repeats within viewing duration. Terminal dots undergo independent scale pulse: 1.0 to 1.4 over 126 frames (2.1s at 60fps) using easeInOutQuad, then 1.4 to 1.0 over 126 frames. Opacity of stroke weight fades from 0 to 255 over 20 frames as strand reaches maximum compression, holds for compression duration, then fades 255 to 0 over 30 frames during expansion.

## Rhythm
Macro rhythm is polyrhythmic breathing — seven overlapping cycles with periods ranging 2.8s to 5.0s create interference patterns that feel organic rather than mechanical. The overall composition exhibits a 12s meta-cycle (Phase 3 'catch' event interval), punctuating the continuous flow with momentary suspension. Micro rhythm is the frame-by-frame noise wobble advancing at 0.002 units/frame (36-frame period before noticeable repetition at this frequency). The lateral drift in Phase 2 operates on an 18s sine wave, adding a slower gestural layer beneath the breathing. Stroke appearance/disappearance creates visual accents every 3-4s (varies per strand due to offset cycles), adding staccato punctuation to otherwise smooth motion. Terminal dot pulses at 2.1s intervals create a counterpoint rhythm to the 3.2s average strand breathing, generating syncopation every 6-8s when cycles align.

## Interaction (passive)
Fully autonomous. The animation is driven by a master timer using millis() to track phase transitions at 12s and 35s marks. Each strand has an independent oscillator object tracking its breathing cycle progress (0.0 to 1.0) which advances based on its assigned period. Lateral drift uses millis() % 18000 mapped to sine function for smooth looping. The 'catch' event system uses a random selector that triggers every 12s — at each interval, random(7) selects a strand index, that strand's breathing cycle is interrupted and replaced with the pause-snap-rebound sequence for 84 frames before resuming normal cycle. Noise offsets for edge wobble are stored per-strand and increment continuously (xOff += 0.002, yOff += 0.001 each frame). Terminal dot scales use independent sine oscillators with frequency 1/126 frames, offset by 42 frames (0.7s) from parent strand breathing phase.
